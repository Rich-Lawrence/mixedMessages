# mixedMessages

## Introduction

The brief for this project was to build a message generator. Everytime the project is run the user recieves a different message. The message that the generator outputs should be made up of (at least) 3 different pieces of data.

## Technologies

*JavaScript
*HTML5
*CSS

## Setup

To run this project, download both the .html and the .js files and run the .html file from your computer.
